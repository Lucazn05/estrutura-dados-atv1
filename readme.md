# Exercícios de Estrutura de Dados em C

Lista de exercícios desenvolvidos na disciplina de Estrutura de Dados utilizando a linguagem C.

## Exercícios

- Produtos complementares
- Soma de reproduções consecutivas
- Avaliação de funcionários
- Contagem de vogais e consoantes
- Vetor reverso
- Números maiores que 5
- Letras alternadas
- Idades ímpares

## Tecnologias utilizadas

- Linguagem C
- Visual Studio Code
